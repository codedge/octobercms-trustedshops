# OctoberCMS - Trusted Shops Plugin

This is a [OctoberCMS](https://octobercms.com) plugin to place a [Trusted Shops](http://www.trustedshops.com/) badge
 on your site. It supports the badge, the product reviews and the product ratings through the backend configuration of 
 OctoberCMS.

Ideally you can use it with the [Octoshop](https://octobercms.com/plugin/feegleweb-octoshop) or
 [Octoshop Lite](https://octobercms.com/plugin/feegleweb-octoshoplite) plugin.

## Installation
### Via [OctoberCMS Marketplace](https://octobercms.com/plugins)
1. Go to the backend area of your OctoberCMS installation
2. Click on `Updates & Plugins`
3. Click on `Install Plugins`
4. Search for `Trusted Shops` and install the plugin

Afterwards please make sure you enter your Trusted Shops ID in the `Settings > Misc > Trusted Shops` section.
